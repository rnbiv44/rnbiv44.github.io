1st Annual 2015

Summer Kids Fun Run

When: June 7, 2015

Where: Battlefield High School 15000 Graduation Drive, Haymarket, VA 20186

Course: The Stadium Track

Race-Day Registration: 8:30 a.m. No mail in registration

Entry Fees: $5.00

Make checks payable to: Battlefield High School

Divisions: K-1st, 2nd-3rd, 4th-5th, 6th-8th

Course: The Stadium Track. K-1st will run a ½ mile run (2 laps). All other divisions will run a 1-mile run (4 laps).

Starting Time: First division K-1st will start at 9:00 am followed by other divisions ever 20 minutes.

Awards: Each Age Division Awards to Top 3 Male/Female Runners (Ribbons will be given to Fun Run participants).

Refreshments: Water at start and finish.

Contact info: proctots@pwcs

Proceeds: All proceeds will go to the FBLA club to help support the students, as they will travel to the Nationals Conference this summer.
